Exercise 19.1 — Bootstrap 5 + Sass Project Setup

See _custom.scss and README.md in this folder for the full Sass workflow. The styles on this page simulate the compiled output.

npm + Sass Setup Steps (Exercise 19.1)
  1. Run npm init -y in your project folder.
  2. Install Bootstrap source: npm install bootstrap
  3. Install Sass compiler: npm install sass --save-dev
  4. Create scss/_custom.scss and import Bootstrap after your overrides.
  5. Compile: npx sass scss/_custom.scss dist/custom.css
  6. Link dist/custom.css in your HTML instead of the CDN.
