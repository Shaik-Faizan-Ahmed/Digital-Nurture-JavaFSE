Bootstrap Directory Structure

Below is an explanation of Bootstrap's file structure:

bootstrap/
  • css/ — Contains compiled CSS files: bootstrap.css, bootstrap.min.css, and RTL variants. These style the entire framework.
  • js/ — Contains JavaScript plugin files: bootstrap.js, bootstrap.min.js, and bootstrap.bundle.min.js (includes Popper.js for dropdowns/tooltips).
  • icons/ — Bootstrap Icons SVG library with over 1,800 icons available as SVG files or via CSS class names.
Note: Always use bootstrap.bundle.min.js to include Popper.js automatically for components like modals, dropdowns, and tooltips.
